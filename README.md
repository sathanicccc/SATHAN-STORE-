## 🔥 Key Features

| Feature | Description |
| :--- | :--- |
| **🟢 Live App Search** | Instantly search and find any app or game from across the globe. |
| **🎮 Categorized Browser** | Browse apps by categories: **Games**, **Social**, **System (Tools)**, and **Hacking Tools**. |
| **📱 Dynamic App Modal** | View detailed app descriptions, ratings, and categories via a smooth popup modal. |
| **☁️ Direct Cloud Download** | Download APKs directly from trusted third-party cloud sources (like Google Search/APKPure). |
| **⚡ Premium UI/UX** | A sleek, modern dark theme with neon green accents and smooth animations. |
| **🚀 Auto-Load Apps** | Popular trending apps automatically load when the store is opened. |

## 🛠️ Tech Stack

* **🌐 Frontend:**
  * **HTML5:** Modern structured markup.
  * **CSS3:** Premium Dark Theme, Neon effects, Smooth Animations, Grid Layout.
  * **JavaScript (ES6+):** Async Fetch API for live app data, Dynamic DOM manipulation, Modal logic.
* **🔧 API Integration:**
  * Uses **iTunes Search API** to fetch live Android software data (Package names, Descriptions, Ratings).
* **☁️ Hosting:**
  * Hosted 100% Free on **GitHub Pages**.
